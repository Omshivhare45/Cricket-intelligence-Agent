# 🏏 Cricket Intelligence Agent

An AI-powered cricket analytics and prediction tool built during the GDG Agentic Premier League.

---

## 🚀 Features

* 📊 Real-time match insights and analytics
* 🤖 AI-driven predictions (logic implemented in app)
* ⚡ Fast UI powered by Vite + React
* 📈 Player & team comparison system
* 🔗 API-based or logic-driven backend integration

---

## 🧠 Backend Logic

Although this project is deployed as a frontend app, it includes backend-style logic:

* Data processing and validation inside the app
* Match prediction algorithms
* API handling / async data fetching
* Modular logic inside `src/` (services / utils)

👉 Example structure:

```
src/
 ├── components/
 ├── services/      # API calls / logic
 ├── utils/         # prediction logic
 └── pages/
```

---

## 🛠️ Tech Stack

* Frontend: React (Vite)
* Logic Layer: JavaScript (custom algorithms)
* Deployment: Render / Vercel

---

git clone https://github.com/Omshivhare45/Cricket-intelligence-Agent.git
